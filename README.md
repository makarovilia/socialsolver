# socialsolver
веб-приложение

Чтобы запустить веб-приложение на локальном сервере достаточно(при условии, что у вас утсановлен Django и прочее необходимое ПО, если его нет, установите) вбить команды:

cd myproject

python manage.py runserver

Необходимое ПО:

Django

gunicorn

psycopg2-binary

whitenoise

dj-database-ur

Все их можно установить через pip install

Ссылка на приложение на сервере:https://socialsolver.onrender.com
